# 还我一个土木梦 - 游戏指南

> 版本 v1.4 | 更新日期 2026-02-03

欢迎来到《还我一个土木梦》！这是一款模拟土木工程项目经理职业生涯的H5网页游戏。

---

## 🎮 游戏简介

你从实习生开始，通过处理事件、管理资源、维护关系，最终晋升为合伙人。

**核心循环：** 创建角色 → 处理事件 → 行动点管理 → 季度结算 → 晋升 → 循环

---

## 📊 核心属性

| 属性 | 说明 | 注意 |
|------|------|------|
| 💰 **现金** | 购买材料、维护关系 | < 5,000 破产 |
| ❤️ **健康** | 执行行动的基础 | ≤ 0 过劳猝死 |
| ⭐ **声誉** | 晋升的关键条件 | ≤ 0 行业封杀 |
| 📚 **工作能力** | 解锁隐藏选项、价格预测 | 通过训练提升 |
| 🎲 **幸运** | 解锁冒险选项、特殊事件 | 通过训练提升 |

---

## 🪜 职级系统（6级）

| 职级 | 图标 | 晋升条件 | 特点 |
|------|------|---------|------|
| **实习生** | 🎓 | 入门级 | 只能维护甲方、劳务队关系 |
| **工程师** | 👨‍🔧 | 20万净资产、1个项目、声誉45 | 一次性解锁监理、设计院 |
| **高级工程师** | 👨‍💼 | 100万净资产、6个项目、1个优质项目 | 解锁政府部门 |
| **项目经理** | 📋 | 400万净资产、12个项目 | **开放团队系统** |
| **项目总监** | 🎯 | 1200万净资产、22个项目、5个优质项目 | 全面资源管理 |
| **合伙人** | 👑 | 4000万净资产、35个项目、8个优质项目 | 🎉 胜利！ |

**晋升检查时机：** 每季度结算时

## 🤝 关系系统

五大关系需要定期维护，否则会衰减并触发负面事件。

### 关系类型

| 关系 | 图标 | 初始值 | 衰减率 | 解锁时机 |
|------|------|--------|--------|---------|
| 甲方 | 🤵 | 50 | 4/季度 | 实习生 |
| 劳务队 | 👷 | 50 | 2/季度 | 实习生 |
| 监理 | 📝 | 50 | 3/季度 | **工程师** |
| 设计院 | 📐 | 50 | 3/季度 | **工程师** |
| 政府部门 | 🏛️ | 50 | 5/季度 | 高级工程师 |

### 维护方式

每种关系有 2 种维护方式，风险和收益不同：

- **应酬宴请**（甲方）：高消耗、高风险、高收益
- **项目汇报**（甲方）：低消耗、稳定收益
- **技术交流**（设计院）：额外提升工作能力
- **配合验收**（监理）：有机会自动放水
- **现场慰问**（劳务队）：低成本
- **公关拜访**（政府）：高风险高回报

### 关系警告

- 关系 ≤ 40：红色警告 ⚠️，负面事件风险增加
- 关系 ≤ 30：橙色警告 🚨，可能触发严重负面事件
- 关系 ≤ 15：功能受限，无法正常使用

## 🎯 行动系统

### 行动点机制

```计算公式
最大行动点 = floor(健康 / 20)

健康 100 → 5行动点
健康 60 → 3行动点
```

### 行动类型

**前期行动（实习生 → 高级工程师）：**

| 行动 | 消耗 | 效果 |
|------|------|------|
| 做项目 | 1点行动点 + 5000现金 | 进度+10、质量+5、健康-5 |
| 承接私活 | 1点行动点 | 现金+15000、健康-8、声誉-5 |
| 偷工减料 | 1点行动点 | 进度+18、质量-12、声誉-3 |
| 休息 | 1点行动点 | 健康+12 |

**后期行动（项目经理 → 合伙人）：**

| 行动 | 消耗 | 效果 |
|------|------|------|
| 招募成员 | 1点行动点 + 招募成本 | 添加团队成员 |
| 团队项目 | 1点行动点 | 委派团队执行项目 |
| 解决问题 | 1点行动点 | 处理团队问题 |

## 💼 材料市场系统

### 材料类型

| 材料 | 图标 | 基准价格 | 波动幅度 |
|------|------|---------|---------|
| 水泥 | 🧱 | 45元/吨 | ±20% |
| 钢筋 | 🔩 | 420元/吨 | ±15% |
| 砂石 | ⛰️ | 12元/立方 | ±25% |
| 混凝土 | 🏗️ | 38元/立方 | ±18% |

### 价格预测

**工作能力越高，预测越准确：**

- 工作能力 0 → 准确率 50%（纯猜测）
- 工作能力 50 → 准确率 75%
- 工作能力 100 → 准确率 100%

### 交易规则

- 每季度最多交易 3 次
- 仓储费按季度收取
- 可以低买高卖赚取差价

## 🎲 隐藏选项系统

### 技术选项 🔧

**解锁条件：** 工作能力达到阈值

- 实习生：工作能力 ≥ 20
- 工程师：工作能力 ≥ 30
- 高级工程师：工作能力 ≥ 50
- 项目经理+：工作能力 ≥ 55

**效果：** 更好的解决方案

### 冒险选项 🎲

**解锁条件：** 幸运达到阈值

- 实习生：幸运 ≥ 25
- 工程师：幸运 ≥ 35
- 高级工程师：幸运 ≥ 45
- 项目经理+：幸运 ≥ 55

**效果：** 高风险高回报，可能失败

**成功率公式：**
```
成功阈值 = 100 - 风险系数 × 100 + 幸运 / 2
```

## 🏆 胜利与失败

### 胜利条件 🎉

**晋升到合伙人** - 满足所有要求后自动触发

### 失败条件 💀

| 条件 | 触发阈值 | 结局 |
|------|---------|------|
| 破产 | 现金 ≤ 0 | 资金链断裂，公司宣告破产 |
| 过劳 | 健康 ≤ 0 | 身体垮了，被送进医院 |
| 封杀 | 声誉 ≤ 0 | 行业封杀，没有任何公司愿意合作 |

---

## 📈 季度结算

每季度结束时：

1. **发放工资** - 基础工资 + 可能的涨薪
2. **扣除生活成本** - 工资的 10%-25%
3. **扣除仓储费** - 库存材料 × 仓储费率
4. **关系衰减** - 所有关系自动衰减
5. **晋升检查** - 满足条件则晋升

## 🌟 特殊机制

### 优质项目

**定义：** 项目质量 ≥ 90

**重要性：**
- 高级工程师晋升需要 1 个优质项目
- 项目总监晋升需要 5 个优质项目
- 合伙人晋升需要 8 个优质项目

### 贵人相助

**触发条件：** 幸运 ≥ 60

**效果：** 15% 概率触发
- 免费维护关系
- 或维护收益翻倍

### 价格预测加成

**触发条件：** 关系维护获得特殊效果

- ✨ 设计优化方案：价格预测准确率 +50%
- 📖 政策解读：仓储费 -50%

## 💡 新手建议

1. **优先级：** 现金 > 健康 > 声誉
2. **早期策略：** 快速完成第一个项目晋升工程师
3. **关系维护：** 每季度至少维护 2-3 个关键关系
4. **价格预测：** 有工作能力再大量交易材料
5. **偷工减料：** 慎用，会大幅降低声誉
